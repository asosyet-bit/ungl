# ungl
Данная практика реализовывалась в рамках моего обучения.
Здесь представлено решение следующей задачи: Создание класса для хранения и управления студентами
Решение содержит следующие этапы:
1. Создание студента: Создайте класс Student с полями: name (имя студента), age (возраст студента), major (специальность) и gpa (средний балл студента).
2. Отображение информации: Реализуйте метод display_info(self), который выводит информацию о студенте в удобном формате.
3. Расчет оценки: Добавьте метод calculate_grade(self), который на основе среднего балла студента будет возвращать его оценку: "Отлично", "Хорошо", "Удовлетворительно" или "Неудовлетворительно".
4. Сортировка студентов: Реализуйте функцию для сортировки списка студентов по их среднему баллу (от наивысшего к наименьшему).
